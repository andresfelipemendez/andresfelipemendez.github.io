About me

I'm a software developer with over a decade of experience across game development, engine architecture and performance optimisation. My journey spans indie titles to AAA pipelines, always centred on squeezing every last cycle from the machine. I prefer data to classes, memory to inheritance and elegant code that serves the hardware.

Having spent years refactoring large codebases from object‑oriented sprawl into tight, cache‑friendly systems, I'm currently building my own engine experiments and exploring cutting edge rendering APIs.
