DOD Engine

DOD Engine is a hands-on project developed to put the principles of data-oriented design into practice and enhance my skills in the C language and Direct 3D. This project serves as a testament to my dedication to continuous learning and my ability to improve upon existing knowledge.

At present, I am implementing a pong game with this engine, with plans to build more complex games in the future. The engine features a data-oriented architecture, utilizing Entity-Component-System (ECS) with a single block of memory for storing entities as a list of archetypes. DLL hot reloading capability is also integrated into the system, allowing for potential multi-platform support.

The DOD Engine is built exclusively in C language, utilizing Direct 3D 11 for rendering. Moving forward, I intend to replace the Direct X SDK with a self-developed simple math library.

For a more in-depth understanding of my work process and to view the code, please visit the project on GitHub.
